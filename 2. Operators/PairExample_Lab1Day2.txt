Each Variable is multiplied together to get the Product

	Initialise the Values for each Integers

Find Sum
	Add each of the ints together (num1 + num2 + num3)
	Print out the result for the Sum
	
Find Product	
	Multiply each of the ints together
	Print out the result for the Product
